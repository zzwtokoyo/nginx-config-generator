# Change Log

All notable changes to the "nginx-config-generator" extension will be documented in this file.

## [1.0.0] - 2024-03-29

### Added
- 初始版本发布
- 自动检测 React、Vue、Angular、Next.js、Nuxt.js 等项目类型
- 生成基础 Nginx 配置
- 生成高级配置（HTTPS、反向代理、负载均衡、CORS、限流）
- 配置验证功能
- 部署脚本生成（传统部署 + Docker）
- GitHub Actions CI/CD 配置生成
- 多模板支持（SPA、SSR、API 代理、负载均衡）
- VS Code 预览功能
- 一键复制配置到剪贴板
